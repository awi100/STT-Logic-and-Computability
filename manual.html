Instructions

  • You must input at least one conclusions.
  • Premises will automatically be made true and the conclusion false.
  • Click on literals and operators, give them a T/F value, and select the corresponding rule.
  • If assigning a value to an operator, choose the rule that has the operator type followed by "Operator".
  • T/F value will turn GREEN with a valid rule choice or RED with an invalid rule choice.
  • Values will be PURPLE until a rule is chosen.

Formatting Rules

  • Cannot double space
  • Literals must be characters from English alphabet
  • Literals must be one character
  • Literals must be seperated by an operator
  • There must be the same amount of open and closed parentheses
  • There can only be one top level operator
